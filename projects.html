CypherStore: Password Generation and Storage App

CypherStore is a simple and secure password management application which allows users to generate secure passwords, as well as store and encrypt their login information locally. This application features AES encryption to allow secure storage of login information on device. Passwords are generated using a secure RNG algorithm, ensuring that it's impossible to recreate your passwords.

Goals: Refresh my C# and Visual Studio Skills, Learn about implementing encryption, Create a handy password storage application for personal use.
Technologies Used: C#.NET, Windows Forms, Visual Studio, AES, CPRNG
